## Reporting a Vulnerability

Please **do not** open a public GitHub issue for security vulnerabilities.

Report privately via GitHub: [Security Advisories](https://github.com/mewc/bun-profiler/security/advisories/new)

Or email: **mikecarter771@gmail.com**

I aim to respond within 48 hours and publish a fix within 7 days of confirmation.
